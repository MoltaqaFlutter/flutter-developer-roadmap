# Task 8 — Event Bus System

**Description:**  
Implement an Observer pattern with classes `Event`, `Listener`, and `EventBus`. Support subscribing, unsubscribing, and emitting events.  

**References:**  
[Observer Pattern — Refactoring Guru](https://refactoring.guru/design-patterns/observer)

**Hints:**  
- Maintain a list of listeners in `EventBus`  
- Notify all subscribed listeners when an event is emitted
